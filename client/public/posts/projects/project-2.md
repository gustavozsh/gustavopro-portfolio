## Sobre o Projeto

Sistema inteligente que compara AWS e Google Cloud usando agentes especializados CrewAI e MCP para encontrar a melhor opção para seu workload. O Cloud Cost Agent analisa automaticamente os requisitos de infraestrutura e fornece recomendações detalhadas de custo-benefício entre os principais provedores de nuvem.

## Funcionalidades

- **Análise Comparativa**: Comparação automática de preços entre AWS e Google Cloud
- **Agentes Especializados**: Utiliza CrewAI para orquestrar múltiplos agentes de IA
- **Integração MCP**: Protocolo de Contexto de Modelo para comunicação eficiente
- **Recomendações Inteligentes**: Sugestões baseadas em workload específico
- **Relatórios Detalhados**: Geração de relatórios com breakdown de custos

## Tecnologias Utilizadas

- **CrewAI**: Framework para orquestração de agentes de IA
- **MCP (Model Context Protocol)**: Protocolo de comunicação entre modelos
- **Python**: Linguagem principal de desenvolvimento
- **AWS SDK**: Integração com serviços Amazon Web Services
- **Google Cloud SDK**: Integração com Google Cloud Platform
- **LangChain**: Framework para aplicações com LLMs
